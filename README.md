# Smart Event Hub

## 📌 Overview
Smart Event Hub is a full-stack web application that allows users to discover, book, and manage events efficiently.

## 🚀 Features
- User authentication system  
- Browse and explore events  
- Event booking functionality  
- Secure payments using Razorpay  
- QR code generation for tickets  

## 🛠 Tech Stack
- Backend: Flask  
- Database: MongoDB  
- Frontend: HTML, CSS, JavaScript  
- Tools: Postman
